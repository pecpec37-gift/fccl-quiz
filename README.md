# 💻 FCCLクイズ道場

富士通クライアントコンピューティング（FCCL）に関する4択＋○×ミックスのクイズゲームです。  
ブラウザだけで動作し、インストール不要・サーバー不要です。

## 🎮 遊び方

1. TOPページでジャンルを選ぶ
2. プレイヤー名を入力してスタート（任意）
3. 120秒以内に10問を解く（4択と○×がランダムにミックス）
4. 残り時間がボーナス点に加算！
5. 結果画面で「スコアを登録する」→ この端末にスコアが保存される
6. 「🏆 ランキングを見る」でジャンル別Top20を確認

## ⚙️ ルール

| 項目 | 内容 |
|------|------|
| 制限時間 | 120秒 |
| 出題数 | 各ジャンル10問（ランダム） |
| 形式 | 4択と○×のミックス |
| 正解 | +10点 ＋ 残り時間+5秒 |
| 不正解 | 減点なし・残り時間−5秒 |
| ボーナス | 残り時間がそのまま加算 |

## 📚 ジャンルと問題数

| ジャンル | 問題数 |
|----------|--------|
| 🏢 会社・歴史・沿革 | 25問 |
| 🖥️ FMVシリーズ | 25問 |
| 📟 FMシリーズ（FM-8/7/77等） | 25問 |
| 🎮 FM TOWNSシリーズ | 25問 |
| 💼 FMRシリーズ | 20問 |
| 📱 その他製品 | 20問 |
| **合計** | **140問超** |

## 🏆 ランキングについて

- スコアはこの端末のブラウザ（localStorage）に保存されます
- 同じ端末で複数人が遊ぶ場合、プレイヤー名を変えると各自の記録が残ります
- ジャンルごとにTop20件を保存

## 🚀 GitHub Pages でのデプロイ方法

```
1. GitHubで新規リポジトリを作成（例: fccl-quiz）
2. index.html と README.md をアップロード
3. Settings → Pages → Source: Deploy from branch → main / (root)
4. 数分後に公開URL発行
   → https://your-username.github.io/fccl-quiz/
```
